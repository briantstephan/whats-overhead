# whats-overhead

A simple website to show what planes are flying directly overhead or nearby.

## Getting started

Prereqs: pnpm (recommended), Node 18+.

1. Install deps: `pnpm install`
2. Start dev server: `pnpm dev` (Vite on http://localhost:5173)
3. Lint: `pnpm lint`
4. Format: `pnpm format`
5. Type-check: `pnpm typecheck`
6. Build for static hosting: `pnpm build`
7. Preview built app: `pnpm preview`
